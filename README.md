To use:

1. git clone this repo.
2. go build

```
Usage of ./shadir:
  -dir string
    	directory to process (default ".")
  -exclude string
    	exclude files matching this regex pattern
  -follow-symlinks
    	follow symlinks
  -hash string
    	hash algorithm (default "sha256")
  -poolsize int
    	number of workers (default 8)
```
